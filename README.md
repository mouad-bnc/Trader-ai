# Trader AI V4

Application Streamlit légère pour suivre BTC, SOL, SUI et DOGE via CoinGecko.

## Déploiement Streamlit Cloud

1. Uploade ces fichiers dans ton dépôt GitHub.
2. Dans Streamlit Cloud :
   - Repository : ton dépôt
   - Branch : main
   - Main file path : app.py
3. Clique Deploy.

## Local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Sécurité

Cette version n'utilise aucune clé API Binance et n'exécute aucun ordre.
